## Overview

**Duka Flow** is a modern offline-first retail Point of Sale (POS) and commerce platform built specifically for African SMEs operating in low-connectivity environments.

The platform is engineered to function reliably during:

* internet outages
* Safaricom instability
* power interruptions
* backend failures

Duka Flow combines:

* POS operations
* inventory management
* M-Pesa workflows
* analytics
* synchronization
* auditability

into a unified commerce operating system.

Duka Flow uses an **Inventory Ledger System** instead of directly modifying stock quantities.

Instead of:

```sql
UPDATE products SET quantity = quantity - 1
```

The system stores immutable stock movement events.

Example:

| Type     | Quantity |
| -------- | -------- |
| purchase | +100     |
| sale     | -2       |
| damage   | -1       |
| return   | +1       |

Current balance is computed dynamically.

Benefits:

* auditability
* reversibility
* fraud detection
* accounting integrity
* branch synchronization
* analytics support

# Audit Trails

All critical actions are logged:

* refunds
* void sales
* stock edits
* debt adjustments
* price changes
* logins

Financial records are never deleted.

Transactions are:

* voided
* reversed
